# UI/UX デザイン仕様書 (Design Specifications)

現在実装されている PayMap アプリのUIデザインおよび画面仕様のまとめです。
全体的にiOS標準のUIコンポーネント（Human Interface Guidelines）に準拠した、クリーンで直感的なデザインを採用しています。

## 1. アプリ全体の構成
*   **ナビゲーション**: 画面下部の `TabView` による切り替え。
    *   **左タブ**: マップ (`map.fill` アイコン)
    *   **右タブ**: マイページ (`person.fill` アイコン)
*   **カラーテーマ**:
    *   メインアクセントカラー: **ブルー**（ボタンやリンク、ユーザーのポジティブなアクションに使用）
    *   プレミアム機能カラー: **イエロー / パープル**（王冠やキラキラアイコン等で特別感を演出）

---

## 2. 画面ごとのデザイン仕様

### 2.1. 認証画面 (AuthView)
*   **背景色**: iOS標準のグループ化背景 (`.systemGroupedBackground`)
*   **レイアウト**: 画面中央にアプリアイコンとキャッチコピーを配置し、下部にログインボタンをまとめた構成。
*   **ログインボタン**:
    *   **Apple**: iOSネイティブの `SignInWithAppleButton` (黒スタイル)。
    *   **Google**: 白背景＋グレーの枠線＋黒文字のカスタムボタンで、Appleボタンと高さを統一。

### 2.2. マップ画面 (MapView)
*   **マップ表示**: 画面全体 (`ignoresSafeArea`) に地図を表示。
*   **店舗ピン (StorePinView)**:
    *   赤いピンアイコン (`mappin.circle.fill`) の下に、白い背景の吹き出し（角丸＋シャドウ）で店舗名を表示。
*   **店舗詳細シート (StoreDetailSheet)**:
    *   ピンをタップすると、画面下部からスプリングアニメーションでせり上がるハーフモーダルシート。
    *   店舗名、カテゴリ、および「利用可能な決済手段」を横スクロール (`ScrollView(.horizontal)`) で青色のタグ風に表示。
    *   右側に青色の「追加・編集」ボタンを配置。

### 2.3. 決済手段の登録画面 (PaymentRegistrationView)
*   **表示形式**: 店舗詳細シートからフルスクリーン寄りのモーダルシートとして展開。
*   **レイアウト**: リストビュー (`List`) 形式。
*   **コンセンサスUI**: 各決済手段の右側に、「いいね (👍)」と「ちがうね (👎)」のアイコンボタンを配置。現在有効とされている決済手段は👍が青くハイライトされる。

### 2.4. マイページ (ProfileView)
*   **レイアウト**: リストビュー (`List`) によるセクション分け。
*   **プロフィールセクション**: ユーザーアイコンと名前、メールアドレスを表示。
*   **実績・インセンティブセクション**:
    *   貢献ポイントをアクセントカラー（青）で強調表示。
    *   獲得バッジを横並びで表示（例：黄色の星、オレンジの炎など）。未獲得のバッジは半透明（グレー）で表示して収集欲を刺激。
*   **プレミアム誘導セクション**:
    *   パープルのキラキラアイコンとテキストで、プレミアムプラン詳細への動線を配置。

### 2.5. プレミアムプラン詳細画面 (PremiumFeatureView)
*   **レイアウト**: スクロールビュー。上部に大きな黄色の王冠アイコンを配置し、特別感を演出。
*   **機能リスト**: 各プレミアム機能（広告非表示、オフラインマップ等）を、パープルのアイコン＋タイトル＋説明文のリスト形式でわかりやすく表示。
*   **購入ボタン**: 画面下部に固定（または末尾）で配置される、目立つ黄色のサブスクリプション登録ボタン。

> [!TIP]
> **今後のデザイン改修ポイント**
> 現在はSwiftUIの標準コンポーネントを組み合わせた「ワイヤーフレーム・プロトタイプ」に近い状態です。
> 今後、よりアプリをリッチにするためには以下のようなデザイン拡張が可能です。
> *   ダークモードへの完全対応とカラーパレットの再定義
> *   カスタムフォントの導入
> *   LottieやSwiftUIの `withAnimation` を用いたマイクロインタラクション（ボタンタップ時の心地よい動きなど）の追加
